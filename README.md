# 간단한 쇼핑몰 구현하기

* Back : Spring
* DB : h2
* FrontEnd : HTML + CSS + JavaScript (Vanilla)


# 회원

- [x] 가입 가능
    - [x] 아이디를 입력한다
    - [x] 비밀번호를 입력한다
    - [x] 성별을 입력한다
    - [x] 회원가입시 50000 포인트를 준다.

- [x] 회원 수정 기능
    - [x] 본인이 맞는지 확인하기위해 비밀번호입력
    - [x] 닉네임을 변경할수 있다
    - [x] 비밀변호를 변경할수 있다

- [x] 탈퇴 기능
    - [x] 본인이 맞는지 확인하기위해 비밀번호입력 후 탈퇴을 진행할수있다.

- [x] 조회 기능
    - [x] 회원의 정보를 조회할수있다.
    - [x] 본인이라면 본인의 장바구니를 조회할수있다.

# 상품

- [X] 상품 목록을 생성한다.
- [x] 상품 목록을 보여준다.
- [x] 상품 목록을 조회한다.
- [x] 상품 목록을 검색할수있다.
- [x] 카테고리 별로 목록을 조회한다.

# 구매

- [x] 상품을 구매 할수있다.
  - [x] 원하는 수량만큼 구매할수있다.
- [x] 상품을 구매하면 재고가 줄어든다.

# 장바구니

- [x] 장바구니에 물품을 담을수있다.
- [x] 장바구니에 담긴 물품들을 구매할수있다.
  - [x] 장바구니에 물품의 수량을 정할수있다.
  - [x] 장바구니에서 물품의 가격을 확인할수있다.
  - [x] 장바구니에서 회원 포인트를 확인할수있다.
  - [x] 장바구니에서 회원포인트 - 가격을 할수있다.
  - [x] 장바구니에서 물품을 구입한다
  - [x] 물품을 구입하면 목록이 사라진다

# 에러
- [x] 프론트엔드 에서 에러를 던져줄 handler 생성

## 회원 예외 상황

- [x] 아이디는 영어와 숫자만 가능합니다.
- [x] 아이디는 빈칸을 입력할 수 없습니다.
- [x] 아이디에 공백을 입력할 수 없습니다.
- [x] 아이디는 중복될 수 없습니다.
- [x] 닉네임은 한글,영어,숫자 및 2-8 글자까지 가능합니다.
- [x] 닉네임에 공백을 사용할 수 없습니다.
- [x] 닉네임은 빈칸을 사용 할수 없습니다.
- [x] 닉네임은 중복될 수 없습니다.
- [x] 패스워드에 공백을 입력할수 없다.
- [x] 패스워드에 빈칸을 입력 할 수 없다.
- [x] 패스워드는 9자리 이상으로 입력해야한다.
- [x] 포인트는 음수가 될수 없다.

## 구매 예외 사항

- [x] 구매수량은 최소 1개부터입니다.
- [x] 구매수량은 재고를 초과하여 입력 할 수 없습니다.
