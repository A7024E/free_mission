# 회원

- [x] 가입 가능
    - [x] 아이디를 입력한다
    - [x] 비밀번호를 입력한다
    - [x] 성별을 입력한다

- [x] 회원 수정 기능
    - [x] 본인이 맞는지 확인하기위해 비밀번호입력
    - [x] 닉네임을 변경할수 있다
    - [x] 비밀변호를 변경할수 있다

- [x] 탈퇴 기능
    - [x] 본인이 맞는지 확인하기위해 비밀번호입력 후 탈퇴을 진행할수있다.

- [x] 조회 기능
    - [x] 회원의 정보를 조회할수있다.
    - 회원의 최신 구매를 조회할수있다.
    - 본인이라면 본인의 장바구니를 조회할수있다.
